## 🎯 À propos

Cette série d'exercices a été conçue pour progresser graduellement dans l'apprentissage du C++. Chaque exercice cible un concept spécifique et renforce la compréhension de la programmation orientée objet.

**Objectifs:**
- 📖 Maîtriser la syntaxe et les fondamentaux du C++
- 🏗️ Comprendre la programmation orientée objet (POO)
- 💾 Gérer les ressources et la mémoire
- 🔄 Pratiquer les patterns de conception

---

## 📖 Contenu des exercices

### Niveau 1: Fondamentaux
- Variables et types de données
- Opérateurs et expressions
- Boucles et conditions
- Fonctions basiques

### Niveau 2: Programmation orientée objet
- Définition de classes
- Constructeurs et destructeurs
- Encapsulation et héritage
- Polymorphisme

### Niveau 3: Structures avancées
- Pointeurs et références
- Allocation dynamique
- Collections et vecteurs
- Gestion des exceptions

---

## 🔧 Prérequis

- **Compilateur C++**: GCC (g++), Clang, ou MSVC
- **Standard C++**: C++11 ou supérieur
- **Build tool** (optionnel): Make ou CMake

```bash
# Vérifier l'installation
g++ --version
```

## 📦 Installation

1. **Cloner le repository**
```bash
git clone https://github.com/MOUHAMADOU80/semestre5_serie1_c-.git
cd semestre5_serie1_c-
```

2. **Compiler un exercice**
```bash
g++ -std=c++11 -o exercice1 exercice1.cpp
```

3. **Exécuter**
```bash
./exercice1
```

## 🎓 Résultats d'apprentissage

Après avoir complété cette série, vous maîtriserez:

✅ **Syntaxe C++**
- Types primitifs et composés
- Variables locales et globales
- Portée et durée de vie

✅ **Programmation Orientée Objet**
- Conception de classes
- Constructeurs/Destructeurs
- Héritage simple et multiple
- Interfaces (classes abstraites)

✅ **Gestion mémoire**
- Allocation (new/delete)
- Pointeurs intelligents
- Pas de fuites mémoire

✅ **Collections**
- Vecteurs (std::vector)
- Listes (std::list)
- Maps et Sets

## 🚀 Recommandations

### Pour bien progresser:

1. **Commencez par le début** - Ne sautez pas d'exercices
2. **Comprenez avant de coder** - Lisez les commentaires
3. **Testez vos modifications** - Compilez régulièrement
4. **Débogez avec gdb** - Apprenez à utiliser le débogueur
5. **Documentez votre code** - Écrivez des commentaires clairs

### Compilation avec optimisations:

```bash
# Debug
g++ -std=c++11 -g -o exercice exercice.cpp

# Release
g++ -std=c++11 -O2 -o exercice exercice.cpp
```

## 🐛 Débogage

### Utiliser gdb:

```bash
# Compiler avec symboles de débogage
g++ -g -o exercice exercice.cpp

# Lancer le débogueur
gdb ./exercice

# Commandes utiles
(gdb) break main
(gdb) run
(gdb) step
(gdb) print variable
(gdb) quit
```
